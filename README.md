# HW2 - Simple GenAI Workflow

## Project Overview
This project builds a simple GenAI workflow for drafting customer support responses.

## Chosen Workflow
The workflow I chose is drafting customer support replies for an e-commerce business.

## User
The main user is a customer support employee who needs to respond to customer messages.

## Input
The system takes a customer message as input. This can include complaints, refund requests, delivery issues, or general questions.

## Output
The system generates a professional email response as a first draft. The response should be polite, clear, and helpful.

## Why This Task Matters
This task is valuable because customer support teams handle many repetitive messages every day. Writing replies can take a lot of time, especially when the issues are similar.

A GenAI system can help generate a first draft quickly, so the support staff only needs to review and adjust it. This can improve efficiency and make responses more consistent. However, human review is still important, especially for sensitive or unclear cases.
